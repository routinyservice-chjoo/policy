개인정보처리방침

루티니 (Routiny) - 습관 트래커
📌 핵심 요약
루티니는 광고가 없는 클린 앱입니다.
사용자의 개인정보를 수집하지 않으며, 모든 습관 데이터는 사용자의 기기에만 저장됩니다.

1. 개인정보의 수집 및 이용

1.1 수집하는 정보 (로컬 저장)

루티니는 앱 기능 수행을 위해 다음 정보를 기기 내부에 저장합니다:

항목 목적 보관 장소
습관 이름, 설정 습관 관리 기능 제공 사용자 기기 내부 (Local DB)
습관 완료 기록 통계 및 달력 표시 사용자 기기 내부 (Local DB)

1.2 개인정보를 수집하지 않습니다

루티니는 서버를 운영하지 않으며, 다음 정보를 절대 수집하거나 외부로 전송하지 않습니다:

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 위치 정보 (GPS)
  • 기기 식별 정보 (Android ID, ADID 등)
  • 사용자의 연락처 및 미디어 파일

2. 광고 및 제3자 서비스

🚫 광고 없음 (No Ads)
이 앱은 Google AdMob을 포함한 어떠한 광고 서비스도 사용하지 않습니다.
따라서 광고 식별자(Advertising ID)를 수집하거나 추적하지 않습니다.

우리는 사용자의 데이터를 수집하는 그 어떤 제3자 SDK(Software Development Kit)도 사용하지 않습니다.

3. 데이터의 저장 및 삭제

저장: 모든 데이터는 사용자의 스마트폰 내부에 암호화되어 안전하게 저장됩니다 (Hive Database 사용).

삭제: 별도의 회원 탈퇴 절차가 필요 없습니다. 앱을 삭제하면 기기에 저장된 모든 데이터가 즉시, 영구적으로 삭제됩니다.

4. 앱 권한 (Permissions)

루티니는 기능을 위해 필요한 최소한의 권한만을 요청합니다.

권한 사용 목적 필수 여부
알림 (Notifications) 사용자가 설정한 시간에 습관 알림을 보내기 위함 (로컬 알림) 선택

* 알림 권한을 허용하지 않아도 앱의 기본 기능(기록)은 이용 가능합니다.

5. 아동의 개인정보 보호

우리는 만 13세 미만 아동의 개인정보를 수집하지 않습니다. 앱 자체가 개인정보를 요구하거나 수집하는 기능이 없으므로, 아동이 안심하고 사용할 수 있습니다.
